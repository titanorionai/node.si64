  Check:  docker logs titan-vanguard-unit
  Fix:    Verify Twitter credentials in .env
          Ensure BRAIN/OLLAMA are healthy
          Check network connectivity

Issue: High memory usage
  Check:  docker stats --no-stream titan-vanguard-unit
  Fix:    Increase memory limit in docker-compose.yml
          docker compose up -d --force-recreate titan-vanguard

Issue: Health check keeps failing
  Check:  docker inspect titan-vanguard-unit | grep -A 10 Health
  Fix:    docker logs -f titan-vanguard-unit
          Check for import errors or credential issues
          May need to rebuild image: docker build -f Dockerfile.vanguard...

Issue: Container keeps restarting
  Check:  docker logs --tail 100 titan-vanguard-unit
  Fix:    Look for exceptions or auth failures
          Verify service dependencies are running
          Check available disk space in /
